ML Text Analyzer

Description

This program is fed a photo containing columns of individual English letters. First, it performs matrix manipulations using Numpy to crop each letter down to each individual letter. Then, 40% of the inputted characters are used to train a machine learning algorithm. Afterwards, the model is tested against the remaining 60% of characters. Using this 40/60 training-to-testing ratio, the model achieved 100% accuracy, identifying all characters in the test set correctly.

Language

Written in Python using Scikit-Learn, Numpy, OpenCV, Scipy and Matplotlib
