Kamlim Website

Description

This web application (located at http://kamlim.herokuapp.com) was developed using the MEAN stack and deployed using Heroku. This project serves as the main webpage for a Christian Retreat site and has an administrator page for CRUD operations.

Language

Written in JavaScript, HTML and CSS

Dependencies

  • Mongoose
  • Express
  • AngularJS
  • Node.js
  • Bootstrap
